Responda, para que servem os comandos: Alert: é uma função embutida que exibe uma caixa de diálogo com uma mensagem para o usuário. Essa mensagem pode ser usada para informar o usuário sobre algo importante, como um aviso ou uma mensagem de erro. Let: let permite que você declare variáveis limitando seu escopo no bloco, instrução, ou em uma expressão na qual ela é usada. Isso é inverso da keyword var , que define uma variável globalmente ou no escopo inteiro de uma função, independentemente do escopo de bloco. //: essa tag é ultilizada para deixar uma mensagem no código sem interferiri na estrutura do site ou a do código, e nem no disigner final da página. If: O if significa "se" e é como usamos os operadores lógicos de uma linguagem. Ele condiciona um comando a uma condição qualquer, executando-o apenas se ela for verdadeira. Sua gramática é "if(condição){ faça isso }". Ou seja, chamamos o if, abrimos parênteses e dentro deles inserimos uma condição qualquer. Else: Além disso, após um if, também podemos chamar um else, que significa "senão". A gramática seria "if(condição){ faça isso } else {faça aquilo}", ou seja, se a condição for verdadeira, faça isso, se não (se ela for falsa) faça aquilo.
